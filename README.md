# Fdf

This project is about creating a simplified 3D graphic representation of a
relief landscape
 
## Compilation

Run `make` command to compile the project

## Usage

```
./fdf [map] [c1] [c2] [c3] [c4] [c5]
```
Some maps are given in the repository

The 5 colors are optional
